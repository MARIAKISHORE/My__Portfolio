## ‚ú® Animation Customization

### Adjust Animation Speed

In `styles.css`, look for animation durations:

```css
animation: fadeInUp 0.8s ease-out;    /* Change 0.8s to faster (0.5s) or slower (1.5s) */
animation: float 6s ease-in-out;      /* Change 6s to desired duration */
```

**Speed Guide:**
- `0.3s` - Very fast, snappy
- `0.5s` - Fast
- `0.8s` - Default, smooth
- `1.2s` - Slower, elegant
- `2s` or more - Very slow

### Disable Animations

To disable animations, find them in `script.js` and comment them out:

```javascript
// Comment out animation initiation
// initScrollAnimations();
```

Or remove animation classes in CSS:

```css
/* Comment out animation lines */
/* animation: fadeInUp 0.8s ease-out; */
```

## üêõ Troubleshooting

### Portfolio Won't Open
- Try different browser (Chrome, Firefox, Safari)
- Make sure all files are in same folder
- Try right-click > Open with > Browser

### Styles Not Loading
- Ensure `styles.css` is in same folder as `index.html`
- Check file names are spelled correctly
- Clear browser cache (Ctrl+Shift+Delete)

### JavaScript Not Working
- Ensure `script.js` is in same folder
- Check browser console for errors (F12)
- Try different browser

### Images Not Showing
- Use correct file path or URL
- Ensure image file exists
- Try using absolute URLs instead

### Mobile Layout Broken
- Check viewport meta tag in HTML
- Clear browser cache
- Test on different mobile device
